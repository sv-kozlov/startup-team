# Sources — QA Engineer

Consolidated external sources referenced across this subagent's skills. Organized by priority.

- ISO/IEC/IEEE 29119-3 Software Testing Documentation — https://www.iso.org/standard/56737.html

- BS 7925-2 Software Component Testing Standard — https://www.bcs.org/
