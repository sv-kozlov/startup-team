# Sources — ML Engineer

Consolidated external sources referenced across this subagent's skills. Organized by priority. Priority 1 sources are canonical and override Priority 2–3 when guidance conflicts.

## Priority 1 — ML canon and framework documentation

Authoritative for ML-idiomatic code, model training, evaluation, and serving. Cited in code-family and method-family skills.

- scikit-learn User Guide — https://scikit-learn.org/stable/user_guide.html
- PyTorch Documentation — https://pytorch.org/docs/stable/
- ONNX — https://onnx.ai/
- MLflow Documentation — https://mlflow.org/docs/latest/
- Hugging Face Transformers — https://huggingface.co/docs/transformers
- pandas Documentation — https://pandas.pydata.org/docs/
- NumPy Documentation — https://numpy.org/doc/
- FastAPI Documentation — https://fastapi.tiangolo.com/
- Pydantic Documentation — https://docs.pydantic.dev/

- OpenTelemetry Semantic Conventions — https://opentelemetry.io/docs/specs/semconv/

- Feast Feature Store Documentation — https://docs.feast.dev/
- Apache Airflow Documentation — https://airflow.apache.org/docs/
- Prefect Documentation — https://docs.prefect.io/
